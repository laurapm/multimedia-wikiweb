Curso para principiantes

¿Por dónde empezar?

Todo programador web que se precie debe tener un conocimiento bastante extenso de:

- HTM5: es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto.

HTML únicamente sirve para indicar cómo se ordena el contenido de una página web, para eso hage uso de las etiquetas. Estas etiquetas se identifican como < Inicio de etiquieta > < / Fin de etiqueta >

- CSS: es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado.
CSS viene de las siglas Cascading Style Sheets.
CSS tiene una sintaxis simple y usa un conjunto de palabras clave en inglés para especificar los nombres de las propiedades de estilo.
Una hoja de estilos consiste en un conjunto de reglas, donde cada consiste en:
   - Uno o varios selectores: los cuáles se encargan de declarar a qué etiquetas se le aplican los estilos. Los selectores pueden aplicarse a todos los elementos de un tipo o los elementos seguidos de un atributo, en concreto, identificadores de una clase o clases que anoten múltiples elementos.
   - Un bloque de declaraciones: consiste en una lista de declaraciones unidas. Cada declaración consiste en una propiedad, dos puntos (:) y un valor. En caso de que haya muchos bloques, se usa un punto y coma (;) para separar cada declaración .

-JavaScript (conocido como JS): es un lenguaje de programación interpretado, definido como orientado a objetos y basado en propototipos. Es imperativo y debilmente tipado. Se usa fundamentalmente en el lado del cliente, implementado como parte de un navegador web, para mejorar la interfaz de usuario y en el lado del servidor, para crear páginas web dinámicas. Fue diseñado con una sintaxis similar a C, pese a que adopta nombres y convenciones de Java. Para interactuar con una página web se provee de una implmentación DOM (Document Object Model), que es una interfaz de plataforma que proporciona un conjunto estándar de objetos para representar documentos, un modelo estándar sobre cómo pueden combioonarse los objetos y una interfaz estándar para acceder a ellos y manipularlos.

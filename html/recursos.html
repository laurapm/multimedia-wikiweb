¿Dónde puedo obtener inspiración?

En esta sección podrás encontrar distintos contenidos multimedia donde obtener inspiración para comenzar con tu página web

  • Instagram: Es una red social en la que se pueden encontrar diversas cuentas con píldoras informativas y cómo elaborar algunos componentes como menús, botones, relojes analógicos... Algunas de estas cuentas son:
  • Páginas web de consulta: A continuación se recomiendan algunas páginas web donde encontrar más información sobre problemas comúnes que ocurren a la hora de programar páginas web o que facilitan el proceso de codificación
  • Obtener iconos: Los iconos pueden ser un gran aporte a la aplicación que se desee creaar, a continuación se exponen algunos recursos gratuitos para obtener los iconos que máss se ajusten al diseño de la página
  • Repositorios github: Variedad de repositorios con contenido útil, donde encontrar como son las aplicaciones del "mundo real", aprender técnicas de programación en JavaScript y plantillas de front-end

Curso para avanzados

¿Cómo puedo hacer más profesional mi página web?

El último paso consistiría en aprender a manejar diferentes frameworks. Los frameworks son estructuras conseptuales y tecnológicas de soporte definido, normalmente con artefactos o módulos de software concretos de base para la organización y desarrollo software. Dicho de otra forma, son esquemas conceptuales que simplifican la elaboración de código.

Los frameworks más utilizados de JavaScript son:

  • Angular: es un framework de JavaScript de código abierto, para aplicaciones SPA (Single Page Application, aplicaciones de una sola página) o PWA (Progressive Web App, aplicaciones web progresivas).
    Ofrece una base para el desarrollo de aplicaciones robustas, escalables y optimizadas, promoviendo el uso de buenas prácticas de programación, un estilo de codificación homogéneo y gran modularidad. El desarrollo en Angular aunque puede hacerse haciendo uso de JavaScript, pero es recomendable sustituirlo por TypeScript, que es un superset de JS que ofrece herramientas adicionales como decoradores o tipado estático.
    A continuación, se puede acceder a un manual de Angular y a la documentación oficial

  • React: es una librería de JavaScript focalizada en el desarrollo de interfaces de usuario. También es un gran aliado para crear todo tipo de aplicaciones web, SPA o aplicaciones móviles. Es mantenido por Facebook y la comunidad de software libre. Hace uso de los patrones MVC (Modelo Vista Controlador) o MVVM.

  • VueJS: es considerado un framework progresivo, es decir, con Vue se pueden crear todo tipo de desarrollos manteniendo un rendimiento y experiencia de usuario muy buena. Con Vue se implementa lo que se conoce como arquitectura de componentes. Permite dividir las aplicaciones en bloques con funcionalidades independientes, llamados componentes. Además, estos componentes se pueden apoyar en otros.
    Otra de las ventajas que ofrece son las vistas reactivas, lo que implica que si se modifican los datos de la aplicación, el HTML y CSS se actualizan de manera automática.
